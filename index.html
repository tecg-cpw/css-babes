Codebabes – Intro à CSS

CSS (Cascading Style Sheets ou Feuilles de Style en Cascade en français) est le langage utilisé pour contrôler la présentation des pages web dans le navigateur. Le navigateur applique les déclarations CSS aux éléments sélectionnés pour les styler.

La syntaxe du CSS

Une règle CSS est un ensemble de déclarations CSS associées à un sélecteur.

Les sélecteurs

Le sélecteur sert à cibler l’élément qu’on veut styler dans le code HTML.

Il existe de nombreux types de sélecteurs en CSS :

  • le sélecteur de type ou sélecteur de balise : c’est le plus simple, c’est juste le nom d’une balise. Il cible tous les éléments qui portent ce nom de balise ;
  • le sélecteur d’id, identifé par le symbole #, qui cible un élément sur base de la valeur de son attribut id ;
  • le sélecteur de classe, identifé par le symbole ., qui cible un élément sur base de la valeur de son attribut class ;
  • le sélecteur de pseudo-classe, identifé par le symbole :, qui cible un élément sur base d’une pseudo-classe ;
  • le sélecteur multiple, constitué de plusieurs sélecteurs séparés par l'opérateur , (VIRGULE) qui signifie ET ;
  • le sélecteur contextuel ou sélecteur de descendant, constitué d’un parent suivi de l'opérateur ESPACE suivi d'un enfant, qui se lit de droite à gauche et qui cible un élément descendant d’un élément dans l'arbre logique du document ;
  • le sélecteur universel, identifé par le symbole *, qui cible tous les éléments de la page ;
  • etc.

Le bloc de déclarations

Le bloc de déclarations sert à appliquer le style à l’élément ciblé. Il est délimité par les accolades ({ }) et peut comporter plusieurs déclarations CSS.

Chaque déclaration CSS sert à changer un des paramètres de style (la couleur de texte, la couleur de fond, la taille de texte, …). Elle se termine par un point-virgule (;).

Chaque déclaration se décompose en deux parties :

  • une propriété ;
  • une valeur ;
  • la propriété est séparée de la valeur par le symbole deux points (:).

La propriété

La propriété sert à préciser quel paramètre on veut modifier.

Les propriétés sont des mots-clés du langage CSS (elles sont listées par ordre alphabétique dans le Pocket Reference). Il en existe un grand nombre, par exemple :

  • color pour changer la couleur de texte ;
  • background-color pour changer la couleur de fond ;
  • font-size pour changer la taille du texte ;
  • line-height pour changer la hauteur de l’interligne ;
  • etc.

La valeur

La valeur sert à préciser ce qu’on a choisi pour la propriété.

Il existe un grand nombre de valeurs possibles en CSS, en fonction des propriétés, par exemple :

  • des valeurs de couleur comme red (couleur nommée), ou #FF0000 (couleur héxadécimale), ou rgb(255,0,0) (couleur RVB), ou rgba(255,0,0,.5) (couleur RVB avec une couche de transparence) pour des propriétés qui servent à définir une couleur comme par exemple color ou background-color ;
  • des valeurs de taille (un chiffre) avec une unité comme px ou em ou % (par exemple 12px) pour des propriétés qui servent à définir des dimensions comme par exemple font-size ou width ;
  • etc.

Le modèle de boîte en CSS

Toute balise HTML est une boîte en CSS. Un p est une boîte, un div est une boîte, un h1 est une boîte, un ul est une boîte, un li est une boîte, un a est une boîte, un em est une boîte, …

Les boîtes correspondant aux différentes balises HTML imbriquées dans l’arbre logique du document s’emboîtent les unes dans les autres comme des poupées russes.

Chaque boîte est composée de quatre éléments :

  • une zone de contenu ;
  • un padding (ou marge interne) ;
  • un border ;
  • un margin (ou marge externe).

La zone de contenu

La zone de contenu est la zone qui contient le texte proprement dit, ou l’image s’il s’agit d’une balise qui affiche une image.

Dans le modèle de boîte standard, la taille de la zone de contenu est définie par les propriétés width (pour la largeur) et height (pour la hauteur).

Attention : Fixer la hauteur d’une zone de contenu qui contient du texte (propriété height) est une très mauvaise pratique, à éviter absolument.

Le padding

Le padding, ou marge interne est la zone qui sépare le contenu de la bordure. Il prend la couleur de fond de l’élément.

Le border

Le border est la bordure qui entoure l’élément. On peut définir son style (trait plein, pointillé, …), son épaisseur, sa couleur.

Le margin

Le margin, ou marge externe, assure la séparation entre les éléments. La marge externe est transparente, de sorte qu’elle aura la couleur de son élément parent.

L’équation du modèle de boîte

Dans le modèle de boîte standard, la largeur totale de la boîte se calcule selon l’équation suivante :

Largeur totale visible = border-left + padding-left + width + border-right + padding-right

Largeur totale = margin-left + largeur totale visible + margin-right
